# Coding-challenge

The app uses the following libraries / topics:

  * Coroutines

  * Retrofit2

  * ViewModels

  * Repository pattern

  * Recycler View
   
  * Card view
  
  
# Note : The list displayed consist of all the products but the cubic weight is displayed only for category "Air conditioners" as per the challenge
